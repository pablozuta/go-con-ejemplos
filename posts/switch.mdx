Nuestro primer programa mostrara el clasico mensaje "Hola Mundo". Aqui esta el codigo completo.

```go
package main

import "fmt"

func main() {
    fmt.Println("Hola Mundo")
}
```

```go
$ go run hola-mundo.go
Hola Mundo

$ go build hola-mundo.go
$ ls
hola-mundo    hola-mundo.go

$ ./hola-mundo
Hola Mundo
```
