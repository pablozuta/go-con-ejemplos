Nuestro primer programa mostrara el clasico mensaje "Hola Mundo". Aqui esta el codigo completo.

```go
package main

import "fmt"

func main() {
    fmt.Println("Hola Mundo")
}
```
Para ejecutar el programa , pon el codigo en hola-mundo.go y usa "go run".

A veces vamos a querer crear ejecutables con nuestro programa. Podemos hacer esto usando "go build". Asi podremos correr ese ejecutable directamente.

```
$ go run hola-mundo.go
Hola Mundo

$ go build hola-mundo.go
$ ls
hola-mundo    hola-mundo.go

$ ./hola-mundo
Hola Mundo
```

Ahora que podemos correr y crear programas basicos con Go, continuemos aprendiendo mas acerca del lenguaje.
