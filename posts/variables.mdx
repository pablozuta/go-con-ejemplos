title: Variables
date: 3
---

En Go, las variables son declaradas explicitamente y usadas por el compiler para por ejemplo verificar un correcto uso de types en las llamadas de funciones.

```go
package main

import "fmt"

func main() {

    var a = "initial"
    fmt.Println(a)

    var b, c int = 1, 2
    fmt.Println(b, c)

    var d = true
    fmt.Println(d)

    var e int
    fmt.Println(e)

    f := "apple"
    fmt.Println(f)
}
```
