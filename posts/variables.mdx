En Go, las variables son declaradas explicitamente y usadas por el compiler para por ejemplo verificar un correcto uso de types en las llamadas de funciones.

```go
package main

import "fmt"

func main() {

    var a = "initial"
    fmt.Println(a)

    var b, c int = 1, 2
    fmt.Println(b, c)

    var d = true
    fmt.Println(d)

    var e int
    fmt.Println(e)

    f := "apple"
    fmt.Println(f)
}
```
var declara una o mas variables.

Puedes declarar varias variables de una vez.

Go va a inferir el tipo de variables inicializadas.

Las variables que sean declaradas sin ser inicializadas son _zero-values_ . Por ejemplo,
el _zero-value_ de un integer es 0.

La sintaxis := es una manera mas corta para declarar e inicializar una variable. Por ejemplo en este caso "var f string = "apple".

```
$ go run variables.go
initial
1 2
true
0
apple
```
