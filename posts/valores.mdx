title: Valores
date: 2

---

Go tiene varios tipos de valores incluyendo strings, integers, floats, boolenas, etc. Aca hay unos pocos ejemplos basicos.

```go
package main

import "fmt"

func main() {

    fmt.Println("go" + "lang")

    fmt.Println("1+1 =", 1+1)
    fmt.Println("7.0/3.0 =", 7.0/3.0)

    fmt.Println(true && false)
    fmt.Println(true || false)
    fmt.Println(!true)
}
```
