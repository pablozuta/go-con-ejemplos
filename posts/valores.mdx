Go tiene varios tipos de valores incluyendo strings, integers, floats, boolens, etc. Aca hay algunos ejemplos basicos.

```go
package main

import "fmt"

func main() {

    fmt.Println("go" + "lang")

    fmt.Println("1+1 =", 1+1)
    fmt.Println("7.0/3.0 =", 7.0/3.0)

    fmt.Println(true && false)
    fmt.Println(true || false)
    fmt.Println(!true)
}
```
Strings, las cuales pueden ser concatenadas con el signo +.

Integers y floats.

Booleans, con operaciones booleanas como es de esperarse.
